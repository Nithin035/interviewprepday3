1. Difference between flex row and column?
In flex row the flex items are represented in a row
In flex column the flex items are represented in a column

2.Explain inline , internal and external stylesheet
styling in css can be done in three ways
inline means we use style attribute inside the tag
internal means we use style attribute in the head
in case of external we create a external stylesheet and link that to the html file using link and provide the necessary styling in it

3.Justify-content allows you to do what?
it allows us to distribute the space between and around content items along the main axis of a flex container,and the inline axis of grid container

4.Difference between absolute and relative positioning?
relative positioning means the element is positioned to its normal position
absolute positioning means the element is postioned absolutely to its first positioned element

5.What is grid-template-columns used for? it specifies the number( and the widths ) of columns in a grid layout

6.What is the z-index in CSS?
it is a css property that defines the order of overlapping html elements
elements with higher index will be placed on top and viceversa

7.What is the difference between padding and margin?
a margin is the space around an element's border, while padding is the space between an element's border and the element's content.

8.What is box sizing?
The box-sizing property allows us to include the padding and border in an element's total width and height.

9.What is animation delay?
The animation-delay CSS property specifies the amount of time to wait from applying the animation to an element before beginning to perform the animation.

10.Which property you will use to merge cells vertically in grid?
column-grid property

11.Which property you will use to merge cells horizontally in grid?
align items

12.Explain CSS box model?
The CSS box model is essentially a box that wraps around every HTML element.
It consists of: margins, borders, padding, and the actual content.

13. What is the difference between display and visibility in CSS?
CSS Display none does not render the element on the document and thus not allocating it any space.
CSS Visibility hidden does renders the element on the document and even the space is allocated but it is not made visible to the user.
